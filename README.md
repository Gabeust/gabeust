## 🚀 Proyectos Destacados

### 💪 Rutinea — SaaS para Personal Trainers
Plataforma SaaS completa para personal trainers construida de cero a producción, de forma independiente.  
Permite gestionar clientes, rutinas personalizadas, seguimiento de progreso y métricas corporales.  
Incluye chat trainer-cliente, registro de series por ejercicio, página pública por trainer con planes y contacto vía WhatsApp.  
**3 trainers en producción con clientes reales. Precio: $35.000/mes.**

> 🔗 [Demo en vivo](https://rutinea.com.ar) — [Página pública de ejemplo](https://rutinea.com.ar/t/elitefit)

**Stack:** Java 23 · Spring Boot 4 · Spring Security · JWT · Redis · MySQL 8 · Docker · OCI · React 18 · TypeScript · Vite · Tailwind CSS
